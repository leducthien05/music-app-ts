# 🎵 Music Streaming Web App (TypeScript)

## 📌 Description
Đây là một ứng dụng web nghe nhạc được xây dựng bằng TypeScript, cho phép người dùng nghe nhạc trực tuyến, tìm kiếm bài hát, quản lý playlist và điều khiển phát nhạc (play, pause).

Dự án được thiết kế theo hướng component-based, tối ưu trải nghiệm người dùng với giao diện thân thiện và responsive.

---

## 🛠️ Tech Stack
- TypeScript
- HTML / CSS / SCSS
- JavaScript (DOM manipulation)
- Node.js (nếu có backend)
- REST API (nếu có dữ liệu server)

---

## ✨ Features

### 🎧 Music Player
- Play / Pause nhạc
- Tua nhạc (seek bar)
- Hiển thị thời gian bài hát

### 🔍 Search
- Tìm kiếm bài hát theo tên

### 🎨 UI/UX
- Giao diện responsive (mobile & desktop)
